# Sparrow Spring

A cozy narrative-driven **visual novel** made in **Godot** (GDScript) for
[Narrative Driven Jam #14](https://itch.io/jam/narrative-driven-jam-14).

> 🎮 Play it on itch.io: **[canterbury.itch.io/sparrow-spring](https://canterbury.itch.io/sparrow-spring)** (Windows)

## About

*Sparrow Spring* is a small village that happens to be where Kate — or you — comes
to live a new life. Through your presence in the village, you will learn the
consequences of your own words. In the end, no matter what your destiny holds,
the only thing that matters is your decision.

Built by **Canterbury**, a small cross-disciplinary team (art, narrative, programming).
**Programming: [@Sherneys](https://github.com/Sherneys)**

## Running from source

1. Install [Godot 4.x](https://godotengine.org/download)
2. Clone this repo and open `project.godot` in the Godot editor
3. Press **F5** to run

## Project structure

    scenes/     game scenes
    scripts/    GDScript gameplay code
    prefabs/    reusable scene components
    resources/  data resources
    assets/     art & audio
